# Now playing made with arduino
## what do you need?
- arduino r4 wifi (or esp32/8266)
- display lcd
- navidrome o subsonic compatible server
## How can i flash the code?
- download from the relese the latest .zip for your device (arduino r4 wifi or esp), open on aduino ide and flash it.
